# main-ssl-pages

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库用于存放和展示多个独立的 HTML 页面。这些页面不隶属于任何特定域名或网站，以文件形式直接归档于此，便于查阅、分享和维护。

## 目录结构

```
├── README.md          # 本说明文件
├── pages/             # 页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 页面引用的静态资源（可选）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有独立 HTML 页面均存放于此目录下。
- `assets/`：若页面需要额外资源（样式、脚本、图片等），可统一放置于此。

## 页面归档说明

- 每个 HTML 文件均为独立的页面，可单独访问。
- 页面内容保持原始状态，仅作必要的格式调整。
- 归档不涉及具体域名、网址或商业推广信息。
- 页面命名应简洁明了，建议使用英文小写字母与连字符，例如 `about-project.html`。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/main-ssl-pages.git
   ```
2. 将需要归档的 HTML 文件放入 `pages/` 目录。
3. 如有需要，可在 `assets/` 中添加资源文件，并在 HTML 中引用相对路径。
4. 提交并推送更改。
